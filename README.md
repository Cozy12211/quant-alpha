# Quant Alpha — 量化交易研究实验室

> 目标：建立、完善、升级交易系统，最终实现睡后收入。

---

## 项目结构

```
quant-alpha/
├── strategies/          # 交易策略
│   ├── backtests/       # 回测结果与报告
│   ├── signals/         # 信号生成逻辑
│   └── models/          # 预测模型（ML/统计）
├── data/                # 市场数据（本地，gitignored）
├── research/            # 研究报告、假设验证、笔记
├── systems/             # 交易系统代码
│   ├── core/            # 核心引擎（订单管理、仓位、执行）
│   ├── connectors/      # 交易所/券商接口
│   └── risk/            # 风控模块（止损、仓位控制）
├── notebooks/           # Jupyter 分析笔记本
└── docs/                # 文档（API、部署、复盘）
```

## 核心原则

1. **可验证**：任何策略必须先回测，再小资金实盘，再放大
2. **可复现**：同一策略在不同时间段、不同品种上必须稳定
3. **有风控**：没有止损的策略不是策略，是赌博
4. **睡后收入**：最终目标是自动化运行，减少人工干预

# Quant Alpha 交易日志 - 使用指南

## 文件位置

| 文件 | 路径 | 用途 |
|---|---|---|
| 交易日志网页 | `~/Desktop/交易日志.html` | 日常使用 |
| 更新脚本 | `~/projects/quant-alpha/update_trades.py` | 数据更新 |
| 备份网页 | `~/projects/quant-alpha/trade-log.html` | 备份 |

---

## 数据更新方式（3种）

### 方式1：自动获取（推荐，交易时间）

双击运行脚本，自动从腾讯API获取实时行情：

```bash
cd ~/projects/quant-alpha
python3 update_trades.py
```

### 方式2：手动输入价格

```bash
cd ~/projects/quant-alpha
python3 update_trades.py 85.50
```

### 方式3：联系 CoCo 更新

收盘后发消息给我：「更新行情」

我会用 kimi_finance 查收盘价，帮你更新所有文件。

---

## 每日流程

### 开盘前（9:00-9:30）

1. 双击打开 `交易日志.html`
2. 切换到「交易模式」
3. 完成5项检查清单
4. 查看买卖点建议

### 收盘后（15:00-15:30）

1. 运行更新脚本获取最新收盘价
2. 切换到「复盘模式」
3. 填写今日交易记录和复盘笔记
4. 查看资金曲线和统计数据

## 数据存储

- **交易记录**：保存在浏览器 localStorage
- **收盘价**：我每天用脚本更新 HTML 文件
- **备份**：每次更新自动同步到项目目录

## 紧急止损

1. 立即清仓
